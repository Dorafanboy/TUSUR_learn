В режиме конфигурации терминала:
указание имени хоста
hostname <имя>

Пароль для консоли:
line console 0
password <пароль>
login
login указывает авторизацию

Пароль для привелегированного режима EXEC:
enable password c1$c0

Настройка зашифрованного пароля:
enable secret <зашифрованный пароль>

Шифрация паролей enable и console:
service password-encryption

Настройка баннера:
Эти баннеры предупреждают пользователей о том, что несанкционированный доступ запрещен.
banner motd "Текст баннера"

В привелегированном режиме:
Вывод текущей конфигурации
show running-config

Вывод стартовой конфигурации:
show startup-config

Вывод сведений об IP-адресе, а также о состоянии всех портов и интерфейсов коммутатора:
show ip interface brief

Резервное копирование файла конфигурации в NVRAM:
copy running-config startup-config
или
wr


Вход в режим конфигурации интерфейса:
interface vlan 1

Указание IP-адреса:
ip address 192.168.1.253 255.255.255.0
Активация интерфейса:
no shutdown

Command Prompt (адрес назначения):
ping 192.168.10.10 

+ конфигурирование хоста в IP Configuration


Флеш-память:
Проверка содержимого во флеш-памяти:
show flash
Сохранение файла загр.конф. во флеш памяти:
copy startup-config flash

Вывод сведений об интерфейсе:
Пример:
show interfaces Serial 0/0/0

BW - пропускная способность

Вывод таблицы маршрутизации
show ip route

Вывод краткой сводки по текущим интерфейсам:
show ip interface brief


Шлюзы: шлюз по-умолчанию, шлюз последней надежды
Шлюз - интерфейс для взаимодействия устройств, находящихся в разных узлах.

Сквозное подключение - проверка удалённых устройств 
(устройств, находящихся в разных локальных сетях)

В S1(config) -> назначение шлюза по-умолчанию:
ip default-gateway 192.168.11.1 (шлюз маршрутизатора в локальной сети)

Включение пересылки IPv6-пакетов на маршрутизаторе:
R1 (config) # ipv6 unicast-routing

Переход на режим конф. интерфейса:
R1 (config) # interface GigabitEthernet0/0
Настройка IPv6-адреса (например, в интерфейсе GigabitEthernet0/0):
R1 (config-if) # ipv6 address 2001:DB8:1::1/64
настройка локального IPv6-адреса:
R1 (config-if) # ipv6 address FE80::1 link-local
+ Активация интерфейса:
R1 (config-if) # no shutdown
