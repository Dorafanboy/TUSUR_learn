R(config)# router rip       - Настройка параметров маршрутизации прот. RIP (по-умолчанию при активируется RIP-v1)

Включение маршрута для той или иной сети для того, чтобы сообщить,
какие локальные интерфейсы используются для обмена данными с другими маршрутизаторами,
а также какие локально подключенные сети должны быть объявлены этим маршрутизатором всем остальным маршрутизаторам
R(config-router)# network сетевой_адрес

RIPv1 - протокол классовой маршрутизации для IPv4 (вводя 192.168.1.32 IOS автоматически преобразует в 192.168.1.0)

R(config)# no router rip       - Отключение и удаление настроек протокола RIP


R1# show ip protocols          - Отображение текущих настроек протокола маршрутизации IPv4 |
Проверка сетей, объявляемых в протоколе RIP на маршрутизаторе R1

R1# show ip route | begin Gateway - Отображение таблицы маршрутизации RIP


Маршрутизатор RIPv1 игнорирует поля RIPv2 в записи маршрута.

Включение RIPv2:
R(config)# router rip
R(config-router)# version 2
R(config-router)# ^Z
R# show ip protocols | section Default

Отключение объединения сетей
R(config-router)# no  auto-summary
R(config-router)# end

Используйте команду конфигурации маршрутизатора passive-interface, 
чтобы запретить передачу обновлений маршрутизации через интерфейс маршрутизации, 
но при этом разрешить объявление сети для других маршрутизаторов.
Запрет передачи обновлений маршрутизации:
R(config-router)# passive-interfase {g0/0}
R(config-router)# end
